Download lwalpapers!

To download all the wallpapers; Follow the procedure below:

Go to main website

  • Open the terminal, just copy and run following commands:

    • bash
mkdir lwalpapers
cd lwalpapers
for (( w=1; w<934; w++ ))
do
  curl -LOS "https://raw.githubusercontent.com/whoisYoges/lwalpapers/PicturesOnly/wallpapers/$(printf b-%0.3d.jpg "$w")"
done

    It will download all 933 wallpapers in lwalpapers directory.
